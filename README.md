# BYNIGHTS

BYNIGHTS è uno script Python che automatizza il login a Instagram per il progetto BYNIGHTS.

## Funzionalità

- Apertura automatica della pagina di login di Instagram.
- Inserimento di username e password.
- Gestione base degli errori di login.

## Requisiti

- Python 3.10 o superiore
- [Playwright per Python](https://playwright.dev/python/)
- Un browser supportato da Playwright (es. Chromium)

## Installazione

Clona il repository e installa i requisiti:

```bash
pip install -r requirements.txt
playwright install
